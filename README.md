# Mobile Offer

Mobile-first prototype of an enterprise ecommerce SaaS offer editor.

## Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React DnD
- Lucide icons

## Local Setup

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/` or the URL printed by Vite.

## Checks

```bash
npm run typecheck
npm run build
```

To preview the production build:

```bash
npm run preview
```

## Repository Notes

Generated folders such as `node_modules/` and `dist/` are ignored and should not be committed.
